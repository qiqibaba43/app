# app

## 联系人列表：
- 显示所有联系人的列表，包括姓名、电话号码和头像。
- 允许用户按姓名或其他字段对联系人进行排序。
## 联系人详情：
- 当用户点击联系人时，显示详细信息如电话号码、电子邮件地址等。
- 提供编辑和删除联系人的功能。
- 支持拨打电话
## 联系人搜索：
- 提供搜索功能，允许用户快速查找特定的联系人。
## 联系人分组：
- 允许用户将联系人分组，如家庭、朋友、工作等，并显示分组信息。
## 联系人导入和导出：
- 支持将联系人导出到一个文本文件。
- 允许用户从本地文本文件导入联系人。
## 主题切换：
- 支持能够切换应用主题，配置两套主题。
## 联系人导航：
- 支持字母导航，能够快速定位该字母下的位联系人。
- 支持分组导航，能够快速定位该分组下的联系人。
## 权限管理：
- 确保应用程序请求并正确处理访问联系人数据所需的权限，如读取联系人和写入联
系人数据权限
